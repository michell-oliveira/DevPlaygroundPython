# DevPlaygroundPython

Projeto em Python com ambiente virtual (`venv`) e dependências gerenciadas por `requirements.txt`.  
Este README contém **toda a documentação necessária** para configurar, rodar e manter o projeto em qualquer máquina.

---

## ✅ Pré-requisitos

Antes de começar, você precisa ter instalado no computador:

- ✅ **Python 3.10 ou superior**
- ✅ **Git** (opcional, se o projeto for versionado)

## ✅ Verificacndo instalação e versão Python

```powershell, terminal vsCode ou outro
python --version
git --version
```

## ✅ Configurando ambiente

```powershell, terminal vsCode ou outro
cd CAMINHO_DA_PASTA_DO_PROJETO
python --version
python -m venv venv
.\venv\Scripts\activate
python -m pip install --upgrade pip
pip install -r requirements.txt
python app.py

```
